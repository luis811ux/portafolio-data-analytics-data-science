# Portafolio de Data Analytics & Data Science

Bienvenido a mi portafolio profesional. Aquí presento proyectos desarrollados en análisis de datos, machine learning y visualización, utilizando metodologías reproducibles y enfocadas en la generación de valor para el negocio.

Este repositorio central organiza y documenta mis trabajos más relevantes.

---

## Sobre mí

Soy un profesional en transición hacia el campo de **Data Analytics** y **Data Science**, con experiencia en:

- Análisis Exploratorio de Datos (EDA)  
- Preparación y limpieza de datos (ETL)  
- Visualización analítica  
- Modelado supervisado (Machine Learning)  
- Ingeniería de características  
- Interpretación de métricas y validación de modelos  
- Documentación técnica y estructuras reproducibles para proyectos de datos  

Mi enfoque combina claridad analítica, trazabilidad técnica y orientación a resultados.

### 2. Telecom X – Análisis Exploratorio de Churn (EDA)

**Repositorio:**  
https://github.com/luis811ux/telecom-x-churn-analysis-eda.git

**Descripción breve:**  
Exploración de datos de clientes para entender patrones de evasión, variables más influyentes y características críticas del fenómeno de churn.

### 3. Telecom X – Predicción de Cancelación (Machine Learning)

https://github.com/luis811ux/telecom-x-churn-prediction-ml.git
